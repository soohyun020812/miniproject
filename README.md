# miniproject
### team5

> **프로젝트 소개**
- 카카오 맵 api와 서울 공공데이터를 활용한 내위치 인근 상권 정보 리포트 제공 서비스

> **개발 기간**
- 2023-05-24 ~ 2023-05-26

> **개발 인원**
- 팀장 : 장호찬
- 팀원 : 박석민, 신희진, 이수현, 조영호

> **개발 목표**
- 예비창업자들을 위해 위치기반의 상권분석서비스를 제공하자
